## 1. Введение
Численное интегрирование является фундаментальной задачей вычислительной математики, возникающей в различных областях науки и техники. Метод трапеций представляет собой один из простейших и наиболее широко применяемых методов численного интегрирования. В данной работе рассматривается реализация параллельного алгоритма вычисления двойного интеграла методом трапеций с использованием технологии MPI для распределенных вычислений.

## 2. Постановка задачи
Задача: реализовать параллельный алгоритм вычисления двойного интеграла функции двух переменных методом трапеций.

Вход: структура Integral, которая содержит:
- function — подынтегральная функция f(x, y) = x² + y²
- limits — пределы интегрирования по x и y
- n_i — число узлов интегрирования по каждой координате

Формат выходных данных: вещественное число — приближенное значение интеграла
