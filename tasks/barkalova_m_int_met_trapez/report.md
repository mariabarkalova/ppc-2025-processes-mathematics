## 1. Введение
Численное интегрирование является фундаментальной задачей вычислительной математики, возникающей в различных областях науки и техники. Метод трапеций представляет собой один из простейших и наиболее широко применяемых методов численного интегрирования. В данной работе рассматривается реализация параллельного алгоритма вычисления двойного интеграла методом трапеций с использованием технологии MPI для распределенных вычислений.

## 2. Постановка задачи
Задача: реализовать параллельный алгоритм вычисления двойного интеграла функции двух переменных методом трапеций.
Математическая формулировка: необходимо вычислить значение двойного интеграла:
    ∬f(x,y)dxdy,  f(x,y)=x^2+y^2

Вход: структура Integral, которая содержит:
- function — подынтегральная функция f(x, y) = x^2 + y^2
- limits — пределы интегрирования по x и y
- n_i — число узлов интегрирования

Формат выходных данных: вещественное число — значение интеграла

## 3. Описание базового алгоритма (последовательный)

Последовательный алгоритм реализует прямое вычисление двойной суммы по формуле метода трапеций

## 4. Схема распараллеливания 

1. Process 0: Читает входные данные, рассылает параметры всем процессам
2. Все процессы: Получают общие параметры (пределы, число узлов)
3. Каждый процесс вычисляет частичную сумму для своих узлов
4. Все процессы отправляют свои суммы на Process 0
5. Process 0: Суммирует все частичные результаты, получает итоговый интеграл
6. Рассылка результата: Process 0 рассылает финальный результат всем процессам

## 5. Детали реализации
Архитектура проекта:
- последовательная версия: barkalova_m_int_met_trapez/seq/
- MPI-версия: barkalova_m_int_met_trapez/mpi/
- общий компонент: barkalova_m_int_met_trapez/common/
- тесты: barkalova_m_int_met_trapez/tests/

Коммуникационные операции: MPI_Bcast(рассылка параметров), MPI_Reduce(сбор частичных сумм), MPI_Bcast(рассылка итогового результата)
